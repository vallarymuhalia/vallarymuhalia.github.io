## 🚀 Getting Started

### Customization Guide

1. **Update Your Information**
   - Edit `index.html` and replace placeholder text with your actual information
   - Update social media links in the Contact section

2. **Add Your Projects**
   - Modify the project cards in the "Projects Section" of `index.html`
   - Add project titles, descriptions, links, and technologies

3. **Update Skills**
   - Edit the skills in the "Skills Section"
   - Add your actual programming languages, frameworks, and tools

4. **Customize Colors**
   - Edit the CSS variables in `styles.css` under `:root`
   - Primary color, secondary color, and other theme colors can be changed

### Key Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

## 💡 Tips
